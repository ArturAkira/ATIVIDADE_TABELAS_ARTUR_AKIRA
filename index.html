Atividade - exercício 1

Align

Serve para mudar o alinhamento de um texto ou dos elementos de alguma tabela ou outro componente.

Valign

Similar ao align, o Valin ou vertical-align serve para ajustar o alinhamento vertical dos elementos.

Cellspacing

Ajusta o espaçamento entre as células.

Cellpadding

Ajusta o padding de uma ou mais células.

Rowspan

Mescla as linhas de uma tabela, ou seja, altera a disposição dos elementos que estão alinhados horizontalmente.

Colspan

Similar ao rowspan, porém mescla colunas, ou seja, altera a disposição dos elementos verticalmente.

width

Altera a largura de uma ou mais células.
